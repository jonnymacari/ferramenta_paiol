Instruções para o arquivo CSV

Formato obrigatório: O arquivo deve conter as seguintes colunas (na ordem exata):

  • username - Nome de usuário (obrigatório, único)
  • email - E-mail (obrigatório, único)
  • first_name - Nome (opcional)
  • last_name - Sobrenome (opcional)
  • user_type - Tipo: admin, gestor ou monitor (opcional, padrão: monitor)
  • cpf - CPF (opcional)
  • telefone - Telefone (opcional)
  • password - Senha (obrigatório)

Exemplo de linha:
joao123,joao@email.com,João,Silva,monitor,123.456.789-00,(11) 99999-9999,senha123

Exemplo de arquivo CSV
username,email,first_name,last_name,user_type,cpf,telefone,password
monitor1,monitor1@email.com,João,Silva,monitor,123.456.789-00,(11) 99999-9999,senha123
monitor2,monitor2@email.com,Maria,Santos,monitor,987.654.321-00,(11) 88888-8888,senha456
gestor1,gestor1@email.com,Carlos,Oliveira,gestor,111.222.333-44,(11) 77777-7777,senha789
